# extract-inline-scripts
extract inline scripts

## Installation
```
npm install --global extract-inline-scripts
```

## Usage
```
$ extis [OPTIONS] globs
```

### OPTIONS
* `-c`
    * print the code.
* `-d`
    * analyze the duplicated code.
* `-n [NUM]`
    * print the first NUM lines.
* `-l [NUM]`
    * extract the scripts with NUM lines or more.
